Projet Mercadona
Mercadona est une application web d'information promotionnelle, construite avec Django.

Installation : 

1.Cloner le dépôt GitHub sur votre machine locale.
2.Créer un environnement virtuel et activer-le.
3.Installer les dépendances avec la commande pip install -r requirements.txt.
4.Créer un fichier .env à la racine du projet avec les variables d'environnement nécessaires.
5.Lancer les migrations avec la commande python manage.py migrate.
6.Créer un superutilisateur avec la commande python manage.py createsuperuser.
7.Lancer le serveur de développement avec la commande python manage.py runserver.


Utilisation : 

L'application web Mercadona permet aux utilisateurs de :

Consulter la liste des produits, avec leurs images, leurs description, leurs noms, leurs prix et leurs catégories.
Filtrer les produits par catégorie.
Filtrer les produits si ils sont en pormotions.
Ajouter des articles.
Ajouter des promotions aux articles déjà crées.


Fonctionnalités : 

Gestion des utilisateurs (connexion, déconnexion).
Gestion des produits (ajout, modification).


Contribuer : 

1.Forker le projet.
2.Créer une branche pour votre fonctionnalité (git checkout -b feature/NomDeMaFonctionnalité).
3.Commiter les changements (git commit -am 'Ajout de ma fonctionnalité').
4.Pousser les changements sur votre branche (git push origin feature/NomDeMaFonctionnalité).
5.Ouvrir une Pull Request.
